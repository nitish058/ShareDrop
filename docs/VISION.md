# Throw Vision

## Why Throw Exists

Throw exists because transferring files between devices should be simple.

The project began with a practical problem. Moving files between devices running different operating systems often requires unnecessary effort for a task that should be straightforward.

Throw is built to solve that problem.

The goal is to provide a reliable, straightforward, and efficient way to transfer files directly between devices.

Throw is an open-source project. Anyone can inspect the source code, understand how the system works, report issues, or contribute improvements.

This project is intended to evolve over time. Every version should improve the software through thoughtful engineering and careful design while keeping the user experience simple.
